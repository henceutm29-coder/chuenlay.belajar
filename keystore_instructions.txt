
Keystore & Signing Instructions
--------------------------------
1. Generate a keystore (locally):
   keytool -genkey -v -keystore chuenlay_keystore.jks -keyalg RSA -keysize 2048 -validity 10000 -alias chuenlay_key

2. Place the keystore in android/app/ and set the following in android/key.properties:
   storePassword=<your_store_password>
   keyPassword=<your_key_password>
   keyAlias=chuenlay_key
   storeFile=chuenlay_keystore.jks

3. Configure android/app/build.gradle to load key.properties for signing configs.
4. In GitHub Actions, for production builds, use secrets to provide signing keys or sign locally.
